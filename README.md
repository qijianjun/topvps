# 关于
本repo是[TopVPS的VPS网络速度测试报告](https://vps123.top/pingtests/)的源文件，亦分享VPS常用的API和测速等方面的脚本。

## API文件说明
- vultr.py  |  Vultr的api调用类
- vultr.sh  |  系统初始化脚本，配置ssh、git、vim、python、tmux、shadowsocks等
